ASTR777 Star and Planet Formation Problem Set 2 - STARTER NOTEBOOK#

numerically integrate the Emden-Chandrasekhar equation and plot Bonnor-Ebert density profile#

\[\frac{1}{x^2}\frac{d}{dx}\left(x^2\frac{dy}{dx}\right) = e^{-y}\]

follow the methodology described in the problem set#

expand the differential to derive, $\(2x y' + x^2 y'' = x^2 e^{-y},\)\( where \)y‚Äô = dy/dx, y‚Äô‚Äô = d^2y/dx^2\(. This gives a prescription for \)y‚Äô‚Äô\( which we can use in Taylor expansions, \)\(y(x+\Delta x) = y(x) + y'(x)\Delta x + y''(x) \Delta x^2/2,\)\( \)\(y'(x+\Delta x) = y'(x) + y''(x)\Delta x.\)\( We then step out from \)x=0\( to determine \)y(x)\(. The initial conditions are \)y(0)=y‚Äô(0)=0\( (gravitational potential is zero and symmetric about the origin). We also need \)y‚Äô‚Äô(0)\( which is a bit tricky but we integrate once to determine \)x^2y‚Äô=\int x^2e^{-y}dx\simeq x^3/3\( where the last approximation is for near the origin. This gives \)y‚Äô\simeq x/3\( which implies \)y‚Äô‚Äô(0)=1/3$.
